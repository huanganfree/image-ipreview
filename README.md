# image-preview
一款基于vue的轻量图片预览插件
## 安装
```
npm install image-preview 
```
### 用法
```
import image-preview from 'image-preview'
import 'image-preview/lib/vue-image-preview.css'

```
### 选项
```
| 属性名|  类型 |  描述   | 默认值  |
| url   | String | 图片路径 |      |
| closeOnPressEscape | Boolean | esc键关闭 | false |

```
